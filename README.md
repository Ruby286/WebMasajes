# 💆‍♂️ Web de Masajes – Landing Page

Este proyecto es una **landing page de servicios de masajes** desarrollada como parte de mi portfolio personal. El objetivo fue crear una web sencilla, clara y orientada a la conversión para un negocio local de bienestar.

---

## 🚀 Demo

La web está desplegada en un servidor real para simular un entorno de producción:

👉 [Ver web en vivo]("[Poner el dominio](https://nuriamatos-masajes.netlify.app/)")

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript 

No se han utilizado frameworks ni librerías externas, con el objetivo de mantener el proyecto ligero, rápido y fácil de mantener.

---

## 🎯 Objetivo del proyecto

- Diseñar una web sencilla para un negocio de masajes.
- Priorizar la experiencia de usuario y la claridad del mensaje.
- Optimizar la velocidad de carga y el SEO básico.
- Practicar el despliegue en un entorno real de producción.

---

## 🌐 Hosting y despliegue

Esta es la primera vez que pruebo a desplegar un proyecto en un **servidor real**, en lugar de ejecutarlo únicamente en local.

El hosting se ha realizado mediante una plataforma de despliegue estático, lo que permite:
- HTTPS automático
- CDN global
- Actualizaciones rápidas mediante Git

---

## 📱 Características

- Diseño responsive adaptado a móvil y desktop
- Sección de servicios
- CTA de contacto
- Diseño minimalista enfocado en conversión
- Carga rápida
